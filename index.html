Let's disable JS temporarily in Chrome


  • Select Settings from the three-dot menu icon
  • Scroll to the bottom and click Advanced
  • Click Content Settings in the Privacy and security section
  • Click JavaScript
  • Switch from Allowed (recommended) to Block

With JS disabled, visit the following sites:


  • netflix.com
  • nytimes.com
  • youtube.com
  • Google search
  • bbc.com/news
  • maps.google.com
  • wikipedia
  • twitter

In 2016, it’s okay to build a website that doesn’t work without JavaScript.
